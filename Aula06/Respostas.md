## Respostas dos Exercícios de Data Center

* **Questão 1:** As alternativas corretas descrevem que a norma NFPA 75 exige separação com resistência ao fogo de uma hora , o uso de diversos sensores estratégicos (ionização, fotoelétrico, etc.)  e o uso de gases inertes como FM200 e $CO_2$ para supressão automática.
**Primeira opção** ✓ - A norma NFPA 75 especifica separação com resistência mínima ao fogo de pelo menos uma hora.
* **Questão 2:** **Unidades UPS e grupo motor-gerador**. Estes são os sistemas auxiliares essenciais para a continuidade da alimentação elétrica.
**Segunda opção** ✓ - Unidades UPS e grupo motor-gerador.
* **Questão 3:** **Tier II**. A classificação é definida pela redundância N+1 no sistema UPS, apesar de possuir apenas um caminho único de entrada de energia.
**Segunda opção** ✓ - Tier II (possui redundância N+1 no UPS, mas única fonte de alimentação e caminho único de entrada).
* **Questão 4:** **A afirmação sobre o PUE (Power Usage Effectiveness) está correta**. O índice relaciona o consumo total do site com o dos equipamentos de TI; quanto menor o valor, melhor a eficiência.
**Quarta opção** ✓ - Sobre PUE (Power Usage Effectiveness): quanto menor o valor, melhor a eficiência energética.
* **Questão 5:** **Certo**. Um Autonomous System (AS) deve possuir links de fornecedores variados para garantir a continuidade em caso de falha.
**Certo** ✓ - A estrutura AS deve possuir links de fornecedores variados para garantir redundância e continuidade dos serviços.
* **Questão 6:** **Errado**. A classificação global de um Data Center é determinada pelo menor nível (Tier) entre seus sistemas; se a parte mecânica é Tier 2, a classificação global será Tier 2.
**Errado** ✗ - A classificação global é determinada pelo **menor** nível entre os componentes. Se elétrica é Tier 3 e mecânica é Tier 2, a classificação global será **Tier 2**.
* **Questão 7:** **Errado**. A redundância 2N implica em uma infraestrutura totalmente duplicada (espelhada), e não apenas a adição de um único módulo (que seria N+1).
**Errado** ✗ - Redundância 2N significa ter **o dobro** da capacidade necessária (sistema completo duplicado), não apenas adicionar um módulo. Adicionar um módulo seria N+1.
* **Questão 8:** **Todas as alternativas anteriores**. A norma ANSI/TIA 942 avalia telecomunicações, arquitetura, elétrica e mecânica para a classificação em Tiers.
**Quinta opção** ✓ - Todas as alternativas anteriores (Redundância, Telecomunicação, Arquitetura e estrutural, Elétrica e Mecânica).
* **Questão 9:** **Tier 1 Básico, Tier 2 Componentes Redundantes, Tier 3 Sistema Auto Sustentado (Manutenção Concorrente) e Tier 4 Alta Tolerância a Falhas**.
**Primeira opção** ✓ - Tier 1 – Básico, Tier 2 – Componentes Redundantes, Tier 3 – Sistema Auto Sustentado e Tier 4 – Alta Tolerância a Falhas.
* **Questão 10:** **Um aterramento isolado para o para-raios e outro para o Data Center, eletrocalhas, racks e piso elevado**.
**Segunda opção** ✓ - Um aterramento isolado para o para-raios e outro para o Data Center, eletrocalhas, racks e piso elevado.
* **Questão 11:** **Data Center Tier III ou Data Center Tier IV**. Estes níveis oferecem a redundância necessária para a infraestrutura crítica.
**Quarta opção** ✓ - Data Center Tier III ou Data Center Tier IV (ambos oferecem redundância adequada para infraestrutura completa).
* **Questão 12:** **I, II, III e V**. Estão corretas as exigências de fontes redundantes (I), pé-direito de 3 metros (II), distância de aeroportos (III) e suporte 24x7 (V). A afirmação IV está incorreta, pois o Tier III exige múltiplos caminhos/links.
**Primeira opção** ✓ - I, II, III e V (o item IV está incorreto pois Tier III requer redundância de links com operadoras).
